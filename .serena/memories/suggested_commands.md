# Development Commands

## Essential
- `npm run dev` — Start dev server with Turbopack
- `npm run build` — Production build (must pass after every change)
- `npm test` — Run all tests with Vitest (must pass after every change)
- `npm run lint` — ESLint on src/

## Deployment
- `npm run deploy` — Build and deploy to Cloudflare Workers
- `npm run preview` — Build and preview locally with Cloudflare
- `npm run cf-typegen` — Regenerate Cloudflare env types after binding changes

## Testing
- `npm test` — Run tests once (vitest run)
- `npm run test:watch` — Watch mode (vitest)

## System (macOS/Darwin)
- `git` — Version control
- Standard unix commands available
